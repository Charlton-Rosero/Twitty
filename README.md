# Twitty
Very simple twitter android applicaton using jetpack compose. Backend is only running locally and is not posted here.

Features:
* Sign up
* Login
* Read Post
* Create Post
* Read Comment
* Create Comment
* Delete Post


Third Party Library Used:
* https://github.com/raamcosta/compose-destinations


Dependecies:
* DaggerHilt
* RetroFit
* DataStore
* LifeCycle
* Kotlin Coroutines
